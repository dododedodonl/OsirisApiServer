Osiris TU/e Unofficial API

By kolibri solutions

Welcome to the unofficial Osiris API, developed by Kolibri Solutions.

This API is in development and subject to change.
All responses are in JSON format except for this help page, data is cached for 24 hours after fetching from Osiris

http GET

The endpoints to use:
Endpoint Description
/ This help page
/api/types/ List of all supported types
/api/faculties/ List of all supported faculties
/api/course/<coursecode>/ List of dictionaries with all information for given course, for now only search by coursecode is supported. If a course has multiple timeslots or quartiles multiple entries will be returned.
/api/faculty/courses/<facultycode>/<typecode>/ List of codes of all courses given by specified faculty with specified type. For a list of available types and faculties see above endpoints.
/api/faculty/courses/<facultycode>/level/<level>/ List of codes of all courses of given level. No type is given because levels are only specified for Bachelor College courses. Level goes from 1-3
